# NayePankh Foundation - Premium Web Portal

A fully responsive, custom-engineered corporate website developed for the **NayePankh Foundation** technical internship submission. This project showcases semantic web best practices, smooth interaction architectures, hardware-accelerated fluid CSS animations, and a system-adaptive native dark mode switch.

## 🚀 Key Features Built-In

* **Modular Architecture:** Separation of concerns using dedicated styles (`style.css`, `responsive.css`, `animations.css`) and modular tracking engines.
* **Aesthetic UI System:** Built with Glassmorphism styles, soft lighting depths, modern typography configuration curves, and precise color scheme variables.
* **Optimized Performance Interactions:** Zero dependence on third-party frameworks or bulky packages like jQuery, keeping resource footings extremely small.
* **Full Accessibility Engine:** Uses appropriate semantic elements, descriptive metadata properties, and keyboard-aware modal/accordion aria roles.

## ⚙️ Direct Setup & Execution Path

1. Extract the package folder onto your local environment workspace.
2. Ensure you drop your custom target visuals directly into the dedicated asset maps matching these exactly:
    * `assets/images/logo.png` (Used across Navbar Header and Master Footer segments)
    * `assets/images/hero-bg.jpg` (Used for full viewport Hero cover tracking background)
3. Launch `index.html` within any modern web browser platform directly to interact with the responsive interface.
